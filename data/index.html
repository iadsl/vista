Data Resources

A collection of crime data sources for spatiotemporal analytics and research

1. Crime Data

City Description
Chicago: Chicago Data Portal | Crimes 2001-Present Reported crimes since 2001 from the Chicago Police Department.
New York City: NYC Open Data | Historic Data (2006–2019) | Current Data (2020–Present) Historical and real-time NYPD crime complaints.
Los Angeles: LA Open Data | Crime Data (2010-Present) LAPD crime data covering various crime types since 2010.
Portland: Portland Police Open Data | Calls for Service | Geospatial Data Crime incidents and dispatch call data.
Detroit: Detroit Open Data | Reported Crimes | Crime Viewer Detroit Police crime reports and public safety insights.
Grand Rapids: GRPD Data Police department reports on crime trends and calls for service.
UK Crime Data: UK Police Data | Crime Data Police crime records from England, Wales, and Northern Ireland.

2. Supporting Datasets for Crime Forecasting

2.1 Points-of-Interest (POI) Data

Dataset Description
Google Places API: Google Places API POI data including bars, schools, transit stations, and more.
OpenStreetMap (OSM) POI Data: OpenStreetMap Detailed urban infrastructure and business location data.
SafeGraph POI Data: SafeGraph Data High-quality business location and visitor behavior data.

2.2 Traffic and Mobility Data

Dataset Description
Uber Movement: Uber Movement Provides traffic congestion patterns in major cities.
Google Traffic Data API: Google Maps Traffic API Real-time and historical traffic flow data.
US Department of Transportation: DOT Open Data National highway and urban mobility data.

2.3 Socioeconomic and Demographic Data

Dataset Description
US Census Data: Census Data Includes income, education, and employment statistics.
American Community Survey (ACS): ACS Data Yearly socioeconomic insights.

Dataset Description

2.5 Public Transportation and Accessibility Data

Dataset Description
